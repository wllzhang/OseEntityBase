航点实体

用于路线规划与点标绘的基础实体，支持显示序号标签，并可结合 航线生成功能（线性/贝塞尔）使用。

overrideprivatevirtual

创建航点实体的渲染节点

创建流程：

  1. 检查 MapNode 是否已绑定（必须）
  2. 创建 CircleNode 作为航点标记（红色圆形，固定高度避免地形遮挡）
  3. 创建 PlaceNode 作为序号标签（使用中文字体）
  4. 将两者组合到 Group 节点中
返回
返回包含 CircleNode 和 PlaceNode 的 Group 节点，失败返回 nullptr
注解
航点使用 CircleNode 和 PlaceNode（osgEarth 注解节点）， 这些节点需要 MapNode 引用，因此必须通过 setMapNode() 预先绑定。 航点的位置通过 GeoPoint 设置，不使用 PAT 节点。
